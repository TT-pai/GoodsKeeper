# GoodsKeeper

你的哆啦A梦好物口袋，装满心动宝藏

AI驱动的个人好物管理工具，采用完全本地存储方案，零云服务成本。

## 功能特性

- 🤖 AI辅助输入：粘贴电商链接一键提取商品信息
- 🔍 智能搜索：关键词搜索、分类浏览、标签筛选
- 🎨 美观分享：一键生成分享卡片PNG
- 💾 本地存储：数据完全本地化，隐私最优
- 📊 数据统计：记录频率、分享次数等行为数据追踪

## 技术栈

React 18 + TypeScript + Ant Design + IndexedDB + Claude API + Canvas

## 快速开始

```bash
npm install
npm run dev
```
